# 8-bit Custom Silicon Calculator (Sky130 ASIC Tape-Out)

A complete RTL-to-GDSII digital physical design of an 8-bit calculator using the open-source **OpenLane compiler** and the **SkyWater 130nm PDK**. The project synthesizes, floorplans, places, routes, and passes all physical verification checks for manufacturing.

## 🚀 Project Overview
This project takes a behavioral Verilog description of an 8-bit arithmetic logic unit (ALU) calculator and compiles it into a fully routed physical silicon layout ready for tape-out.

* **Technology Node:** SkyWater 130nm (sky130A)
* **Standard Cell Library:** sky130_fd_sc_hd
* **Target Frequency:** 100 MHz (Clock Period: 10.0ns)
* **Die Area:** 300 µm × 300 µm

## 🛠️ How to Replicate the Build
To reproduce this GDSII blueprint locally, ensure you have Docker and OpenLane installed, then run:

```bash
# Clone the repository
git clone [https://github.com/YOUR_USERNAME/Calculator-ASIC.git](https://github.com/YOUR_USERNAME/Calculator-ASIC.git)
cd Calculator-ASIC

# Run the OpenLane flow (Assuming OpenLane environment is active)
./flow.tcl -design Calculator
