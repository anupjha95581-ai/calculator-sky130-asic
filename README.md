A complete RTL-to-GDSII digital physical design of an 8-bit synchronous calculator processor. This project drives a custom behavioral Verilog architecture through the open-source **OpenLane compiler** utilizing the **SkyWater 130nm open-source PDK** (`sky130A`). 

The hardware design successfully synthesizes, floorplans, places, routes, and clears all sign-off timing and manufacturing rules (DRC/ERC) for tape-out readiness.

---

## 🛠️ Replication Instructions

Want to reproduce this physical blueprint on your machine? Follow these steps to execute the automated silicon compilation pipeline.

### 📋 Prerequisites
Ensure your host machine has **Docker** and the **OpenLane** environment installed and properly path-exported.

### 1. Clone the Architecture
```bash
git clone [https://github.com/YOUR_USERNAME/calculator-sky130-asic.git](https://github.com/YOUR_USERNAME/calculator-sky130-asic.git)
cd calculator-sky130-asic
# Copy files to the OpenLane design repository
cp -r . /path/to/your/OpenLane/designs/Calculator

# Navigate into your main OpenLane workspace directory
cd /path/to/your/OpenLane/

./flow.tcl -design Calculator

designs/Calculator/runs/RUN_<TIMESTAMP>/results/final/gds/calculator_top.gds
